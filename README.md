## 🛠 1. 개발 환경 (Development Environment)

* **Language:** C++
* **IDE:** Visual Studio 2022, Rider
* **Architecture:** Manager Pattern (Game, Scene, Event)
* **Target Platform:** Windows Console (x64)

---

## 🌟 2. 주요 시스템 기획 (Game Systems)

### 👤 캐릭터 생성 및 육성
* **커스터마이징:** 이름 지정 및 성별 설정.
* **특성 시스템:** '금수저', '잘생긴 외모' 등 게임 시작 시 유리하거나 독특한 효과를 부여하는 특성 선택.
* **성향 시스템:** 간단한 선택지를 통한 캐릭터 성향(Personality) 결정.
* **스탯 투자:** 레벨업 시(최대 Lv.10) 자동 스탯 상승 외에도 유저가 직접 투자할 수 있는 포인트 시스템 제공.

### ⚔️ 전투 및 몬스터
* **단계별 성장:** '수능'을 전투 튜토리얼로 시작하여 학부 과정의 과목들을 몬스터로 형상화.
* **가변 난이도:** 플레이어의 포트폴리오, 어학 점수, 자격증 보유 상태에 따라 최종 보스('취업')의 난이도가 실시간으로 변동.
* **학습 몬스터:** C++, 그래픽스, 서버 등 실제 전공 과목 기반의 몬스터 데이터 설계.

### 🎲 이벤트 매니저 (EventManager)
* **확률 기반 발생:** 긍정적/중립적/부정적 이벤트가 랜덤하게 발생하여 게임의 변수 창출.
* **선택지형 이벤트:** 친구와의 만남 등 상황에 따른 선택 결과가 스탯이나 아이템에 영향.
* **경제 요소:** 주식 투자 등 부가적인 자산 증식 시스템 포함.

### 📊 시스템 로그 & UI
정보의 중요도에 따라 색상별 로그를 출력하여 가시성을 확보합니다.
* ⚪ **INFO (White):** 아이템 획득, 전투 종료 정보.
* 🟡 **WARNING (Yellow):** 랜덤 이벤트 발생, 확률 계산 결과.
* 🔴 **FATAL (Red):** 게임 오버, 시스템 오류, 의도치 않은 동작 발생.

## 🕹 3. 게임 플레이 흐름 (Game Flow)

1.  **Lobby:** 시작, 불러오기, 환경설정 선택.
2.  **Creation:** 캐릭터 정보 및 특성 설정.
3.  **Main Loop:** 훈련(스탯 강화), 상점 이용, 랜덤 이벤트 조우, 전공 몬스터와의 전투.
4.  **Final Battle:** 최종 보스 '취업'과의 전투 후 결과(Clear/Over) 출력.
5.  **Exit:** `ESC` 키를 통해 상태 확인 및 종료 여부 결정.

## 📝 5. 조작 방법 및 특징
* **실행:** Build.zip을 압축풀고 TeamName.exe파일을 실행
* **상태 확인:** 게임 중 `ESC` 키를 눌러 캐릭터의 현재 스탯과 상태를 확인하고 종료 의사를 결정할 수 있습니다.
* **입력 방식:** 콘솔 기반의 숫자 및 키워드 입력 방식을 지원합니다.
